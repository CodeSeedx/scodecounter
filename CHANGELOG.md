# Changelog

All notable changes to this project will be documented in this file.

## [v0.1.0]

- Initial release
- Scan directories for code statistics (lines, code, comments, blanks)
- Support for 100+ programming languages
- Git commit statistics per author
- Terminal and JSON output formats
- Auto-detect .gitignore patterns
- Cross-platform builds: Linux (amd64/arm64), macOS (amd64/arm64), Windows (amd64)
